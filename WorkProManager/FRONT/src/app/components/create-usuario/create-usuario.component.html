Completa el siguiente formulario para crear un nuevo usuario.

Formulario de creación de usuario

Información Personal
El RUT es obligatorio. El RUT debe contener solo números.
El dígito verificador es obligatorio. Solo se permite un carácter.
El nombre es obligatorio. Solo se permiten letras.
El apellido es obligatorio. Solo se permiten letras.
Información de Cuenta
Ingrese una contraseña válida. Debe tener al menos 8 caracteres.
